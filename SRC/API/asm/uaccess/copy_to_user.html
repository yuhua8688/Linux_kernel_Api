copy_to_user


头文件：

#include <asm/uaccess.h>

注：或在asm-generic/uaccess.h


原形：


static inline long copy_to_user(void __user *to,

const void *from, unsigned long n)

{

might_sleep();

if (access_ok(VERIFY_WRITE, to, n))

return __copy_to_user(to, from, n);

else

return n;

}


作用：

从内核空间向用户空间拷贝数据


参数：

to 用户空间地址

from 内核空间地址

n 拷贝的长度
