ioremap


头文件：

#include <linux/io.h>

实际

#include <asm/io.h>


原形：

#ifndef __arch_ioremap

#define ioremap(cookie,size) __arm_ioremap(cookie, size, MT_DEVICE)

#else

#define ioremap(cookie,size) __arch_ioremap((cookie), (size), MT_DEVICE)

#endif


作用：

I/O空间地址重映射


返回值：

重映射后的内存地址的指针


参数：

cookie 开始的地址

size 映射和长度
