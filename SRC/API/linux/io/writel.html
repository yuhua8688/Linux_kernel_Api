writel


头文件：

#include <linux/io.h>

实际

#include <asm/io.h>


原形：

#ifdef __mem_pci

#define writel(v,c) __raw_writel((__force __u32) \

cpu_to_le32(v),__mem_pci(c))

#elif !defined(readb)

#define writel(v,c) __readwrite_bug("writel")

#endif /* __mem_pci */


作用：

将值v写入寄存器c、


参数：

v 要写的值

c 寄存器
