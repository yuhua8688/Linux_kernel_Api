struct miscdevice xxx;


头文件：

#include <linux/miscdevice.h>


原形：

struct miscdevice {

int minor;

const char *name;

const struct file_operations *fops;

struct list_head list;

struct device *parent;

struct device *this_device;

const char *nodename;

mode_t mode;

};


作用：

声明定义杂项设备


成员：

minor 次设备号

name 设备名字

fops 文件操作集
